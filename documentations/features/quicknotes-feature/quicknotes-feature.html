Features

Quick Notes

The “Quick Note” is a floating button in the bottom right corner of the website. The button opens a pop-up when clicked, which allows the users to write a note with any title similar to the notes applications. We also have a create task button in the note, which will create a task with a given title and description when clicked by the user. We can add the task by clicking on the ‘+’ icon, edit it by clicking on the note and delete it by clicking on the trash bin icon on the right of the note.
